ПРОГРАМНОЕ ОБЕСПЕЧЕНИЕ

Ключевые понятия

Програмное обеспечение (ПО)- это программа или множество программ, используемых для управления компьютером.

Типы программного обеспечения (ПО)

    1. Системное програмное обеспечение(süsteemitarkvara, system software)

      2. Программы, принадлежащие к системному ПО, поддерживают работу приложений без учета специфики приложения. Системное ПО поддерживает работоспособность всей системы компьютера.

      Функции системного программного обеспечения

      • управление взаимодействием с оборудованием
      • синхронизация процессов
      • управление памятью
      • общие функуции и работа пользовательского интерфейса, если ни одно из приложений не активно.
    2. Прикладное программное обеспечение (rakendustarkvara, application software)

      3. Задачей прикладного программного обеспечения является выполнение желаний пользователя, поддержка пользователя во имя достижения его целей. Прикладное программное обеспечение опирается на системное программное обеспечение.

      Классификация прикладного программного обеспечения

      • По типу
        • программное обеспечение общего назначения
        • Текстовые редакторы и процессоры
        • Графические редакторы
        • СУБД
        • Электронные таблицы
        • Веб-браузеры
        программное обеспечение развлекательного назначения
        • Медиапрлееры
        • Компьютерные игры
        программное обеспечение специального назначения
        • Эксперементные системы
        • Трансляторы
        • Гипертекстовые системы
        • Системы управления содержимым
        профессиональное програмное обеспечение
        • САПР
        • АСУ
        • АРМ
        • АСУ ТП
        • CRM
      • По сфере применения
        • Прикладное ПО предприятий и организаций.
        • ПО, обеспечивающее доступ пользователя к устройствам компьютера.
        • ПО инфраструктуры предприятия.
        • Образовательное ПО.
        • Имитационное ПО.
        • Прикладные программы для проектирования и конструирования.
      • По типу запуска
        • С установкой
        • Без установки

Аббревиатура

HTML - стандартизированный язык гипертекстовой разметки документов для просмотра веб-страниц в браузере.

CSS - формальный язык описания внешнего вида документа (веб-страницы), написанного с использованием языка разметки (чаще всего HTML или XHTML). Также может применяться к любым XML-документам, например, к SVG или XUL.

JS - ультипарадигменный язык программирования.

Список определений

Файл
Именованная область данных на носителе информации, используемая как базовый обьект взаимодействия с данными в операционных системах.
Папка
Экранный объект в графических интерфейсах операционных систем и программ, дающих доступ к каталогу файловой системы.
Ярлык
Файл, служащий указателем на объект, программу или команду.
